# 筑波大学用語辞典

本リポジトリでは、**情報科学類新入生歓迎パンフレット**に掲載することを目的として、
筑波大学特有の用語や言い回しを **[CSV ファイル](main.csv)** にまとめています。

## 皆さまのご協力を募集しています

今年度の情報科学類新入生歓迎パンフレット担当者は、
筑波大学ならではの用語に十分精通しておらず、**皆さまのお力を必要としています**。

情報科学類に限らず、**他学類・他組織の方からのご提供も大歓迎**です。
内容が事実に基づいていれば、どのような切り口の PR でも構いません。

また、以下のような貢献も歓迎しています。

- 既存用語の説明の修正・改善
- 用例（例文）の追加
- より分かりやすい表現への書き換え

**筑波大学らしいユーモアのある用語**も、ぜひお寄せください。

## WEB 版について

より多くの方に関心を持っていただけるよう、
本用語辞典の **WEB 版を試験的に制作**しています。

## ディレクトリ構成

```text
/
├── CONTRIBUTING.md       <- 貢献方法の案内
├── web                   <- WEB版のソースコード
└── main.csv              <- 用語集（CSV形式）
```
