# 筑波大学用語辞典

本リポジトリでは、**情報科学類新入生歓迎パンフレット**に掲載することを目的として、
筑波大学特有の用語や言い回しを **[CSV ファイル](main.csv)** にまとめています。

## 皆さまのご協力を募集しています

今年度の情報科学類新入生歓迎パンフレット担当者は、
筑波大学ならではの用語に十分精通しておらず、**皆さまのお力を必要としています**。

情報科学類に限らず、**他学類・他組織の方からのご提供も大歓迎**です。
内容が事実に基づいていれば、どのような切り口の PR でも構いません。

また、以下のような貢献も歓迎しています。

- 既存用語の説明の修正・改善
- 用例（例文）の追加
- より分かりやすい表現への書き換え

**筑波大学らしいユーモアのある用語**も、ぜひお寄せください。

## WEB 版について

より多くの方に関心を持っていただけるよう、
本用語辞典の **WEB 版を試験的に制作**しています。

## ディレクトリ構成

```text
/
├── CONTRIBUTING.md       <- 貢献方法の案内
├── web                   <- WEB版のソースコード
└── main.csv              <- 用語集（CSV形式）
```

## `web`の開発環境の構築方法

WEB 版の開発環境を構築するには、以下の手順を実行してください。

- 非Nixユーザの場合

1. `bun`をインストールします。

2. `web`ディレクトリに移動します。

3. 依存関係をインストールします。
   ```sh
   bun install
   ```

4. 開発サーバーを起動します。
   ```sh
   bun dev
   ```

- Nixユーザの場合

flakeの`devShell`を利用して開発環境を構築します。

また、`nix-direnv`を利用できるようにしておくと便利です。
(`nix-ditenv`が導入されている前提で、環境を用意しています。)
